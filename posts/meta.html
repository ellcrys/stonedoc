Meta Block
This is the header of a stone. It contains information about the stone instance and it's issuer.
Structure of a meta block
			
{
   "id":			"jh88hg3sxn43isjhdg0jsdjie03cu3",
   "type":  		"some-coupon",
   "created_at": 	1458623859
}
id (type: string, required: yes): This is the unique id of the stone. It must be a 40 characters long. Preferrable a SHA1 hash of a UUID. This is used by the host and the issuer to recognize a stone instance.

type (type: string, required: yes): This is the unique name of the asset. The asset must be known to the host to allow for proper verification.

created_at (type: number, required: yes): A unix timestamp indicating when a stone instance was created.
