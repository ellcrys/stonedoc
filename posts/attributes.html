Attributes Block
This is an optional block provided for issuers to include arbitrary data into the stone. Data included can be of any JSON data type. This block must be signed by the isser of the asset.
Structure of an ownership block
			
{
   "ref_id":  "jh88hg3sxn43isjhdg0jsdjie03cu3",
   "data": "data of any JSON type"
}
ref_id (string): This is the id of the stone (meta.id) where the ownership block is included. If the value is not the equal to meta.id, the block is considered invalid.

data (any): This field holds any issuer data. The value type can be string, number, JSON object or an array.
