Attributes Block
This is an optional block reserved for issuers to include arbitrary data relevant to their applications. Data included can be of any JSON data type. This block must be signed by the isser of the asset.
Structure of an ownership block
			
{
   "ref_id":  "jh88hg3sxn43isjhdg0jsdjie03cu3",
   "data": "data of any JSON type"
}
ref_id (type: string, required: yes): This is the id of the stone where the ownership block is included. If the value is not equal to meta.id, the block is considered invalid.

data (type: object, required: yes): This field holds the issuer's application specific data. The value type can be string, number, object or an array.
